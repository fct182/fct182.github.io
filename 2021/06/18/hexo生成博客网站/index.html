Hexo+GitHub构建个人博客

需要环境：

Node Git hexo

1.安装Nodejs

2.返回blog目录

3.Github

创建一个新的仓库 YourGithubName.github.io

4.配置_config.yml

5.部署到Github仓库里

​ https://YourGithubName.github.io/ #访问这个地址可以查看博客

6.更改主题Theme

修改hexo根目录下的 _config.yml 文件 ： theme: yilia

https://YourGithubName.github.io/ #查看博客
